
--------------------------------------------------------------------------------
                             Username Policy
--------------------------------------------------------------------------------

The Username Policy module allows site administrators to configure policies for
username, so that every user will have an altered username generated by module.

Project homepage: http://drupal.org/project/username_policy


Installation and Configuration
------------------------------

 * Copy the whole username policy directory to your modules directory
   (e.g. DRUPAL_ROOT/sites/all/modules) and activate the Username Policy module
 * Set permissions at
   http://yoursite.com/admin/people/permissions#module-username_policy
 * The administrative user interface can be found at
   http://yoursite.com/admin/config/people/username_policy
 * Visit http://yoursite.com/admin/help/username_policy for help.
